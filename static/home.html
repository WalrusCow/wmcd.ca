Play.

Once upon a time, I wrote three basic games using the HTML5 canvas in order to teach myself JavaScript. You can play them here - I hope you like them. You can also find the source to each on my GitHub page.

Tetvas Snake Life

Tetvas

It's just a Tetris clone. I created it with my favourite keybindings - sorry if they don't match your expectations!

Use ↑ and Z to rotate, shift to hold and ctrl to drop. Press P to pause the game.

Check out the source on GitHub.

Please use a modern browser to see it in action.

Snake

Most people have played snake before in some form or other. If you don't know how to play, it's very simple. Using the arrow keys (or WSAD) to control the "snake" (orange thing), collect the food (blue things) to grow longer.

You lose if the snake hits itself or any wall!

You can find the source on GitHub.

Please use a modern browser to see it in action.
