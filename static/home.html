Snake

Most people have played snake before in some form or other. If you don't know how to play, it's very simple. Using the arrow keys (or WSAD) to control the "snake" (orange thing), collect the food (blue things) to grow longer.

You lose if the snake hits itself or any wall!

You can find the source on GitHub.

Please use a modern browser to see it in action.
